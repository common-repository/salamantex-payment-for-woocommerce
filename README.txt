== Installation ==

= Minimum Requirements =

* WooCommerce >= 3.6.4
Further minimum requirements are determined by the requirements for WooCommerce:
* PHP 5.6 or greater
* MySQL 5.6 or greater
* WordPress 4.4+
* WP Memory limit of 64 MB or greater (128 MB or higher is preferred)

Furthermore, you need a Salamantex partner account. Please send an email to sales@salamantex.com or [visit our Website.](https://www.salamantex.com/en/client-request).

= Automatic Installation =

1. Log into your WordPress installation.
2. Go to the menu item *Plugins* and then to *Install*.
3. Search for *Salamantex Payment for WooCommerce*.
4. Click *Install Now* and wait until WordPress reports the successful installation.
5. Activate the plugin. You can find the settings here: *WooCommerce => Settings => Checkout => Salamantex Payment for WooCommerce*.

= Manual Installation =

- Download latest plugin from here http://docs.salamantex.com/woocommerce-plugin/
- Install the plugin in Wordpress => Plugins => Add New
- Enable / Activate *Salamantex Payment for WooCommerce*
- Open the setting of the plugin - Enter the *Live Webshop API Key*
- Remove the checkmark from *Testmode*
- Optional: Change title / description of the plugin
- Optional: Decide of you want to wait till the transaction is stable, with the *Bestellabwicklung* Checkbox
- Check that the payment plugin is activated in WooCommerce => Settings => Payment
- Make a testpayment


== Frequently Asked Questions ==

= How to start using the service and How does the installation look like? =

A contract is usually handed over by the mediating distributor and forwarded to Salamantex. If you are interested in using our products or services, please send an email to sales@salamantex.com. One of our sales partners will then contact you as soon as possible. The installation of the service is either carried out by yourself ("self-installation") or on site by a service or sales representative.

= In which countries can the crypto payment service software be used? =

Different legislations regarding virtual currency services in the respective countries make individual assessments per country in which we want to offer our service inevitable for us. Currently, our service is available in the following countries:

- Austria
- Malta
- Croatia

= Which cryptocurrencies can be used to pay? =

Currently, it is possible to pay with the following cryptocurrencies:

- Bitcoin
- Ethereum
- Dash
- Nimiq
- Litecoin
- Ripple
- Stellar

= How to use the Crypto Payment Service? =

The Salamantex Crypto Payment software can be used on all common devices (notebook, PC; tablet, mobile phone) via the internet browser. In stationary trade you can also use the software on our Salamantex Crypto Payment Terminal or on Ingenico terminals of the Tetra Telium series (models Move5000, Lane500, and Desk5000).

In addition, the service can also be easily integrated into your online shop via API or plug-ins. We are open to the possibility of programming or modifying individual software to integrate it into existing hardware systems. If you are interested, please contact us at info@salamantex.com.
